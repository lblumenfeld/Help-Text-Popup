<div><strong>Overview</strong></div>
<div>This is an extension that displays help text as a popup. The user can click a help icon (a question mark) to display a popup box containing help text.</div>
<div>&nbsp;</div>
<div><strong>Help Icon</strong></div>
<div style="padding-left: 30px;">
<ul>
<li>The help icon is a question mark</li>
<li>The icon can be placed anywhere on the sheet&nbsp;</li>
<li>It can be as small as one grid box</li>
</ul>
</div><br>
<div><strong>Help Text Box</strong></div>
<div style="padding-left: 30px;">
<ul>
<li>Overlays the screen content with a box of help content</li>
<li>The box can be any size, up to the height and width of the window</li>
<li>Help text is any HTML content that can be rendered</li>
</ul>
</div>
<div>
